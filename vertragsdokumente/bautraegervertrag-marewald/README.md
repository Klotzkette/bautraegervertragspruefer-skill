## Neu erzeugen

Nach Änderungen an `bautraegervertrag-marewald.md` werden Word- und PDF-Fassung so erzeugt:

```bash
./build.sh
```

Voraussetzungen: `pandoc` und `weasyprint`. Layout und Seitenumbrüche steuern `build/style.css` und `build/pagebreak.lua`.
